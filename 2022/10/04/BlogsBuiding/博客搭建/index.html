博客搭建

发表于 2022-10-04 更新于 2022-10-07 分类于 blogs

前提：默认安装git，node.js以及拥有github账号

大体思路：（1）首先将github和git绑定。（2）使用vue下载hexo框架搭建博客。（3）使用git将初始化后的hexo上传至github。(4)最后介绍一点基本操作。

git与github绑定部分

1.使用git bash的中的ssh生成密钥，指令为

ssh-keygen -t rsa

（ssh生成的密钥用于绑定本地的git和github）

2.windows生成的密钥在C:/Users/你的用户名/.ssh目录下的
id_rsa.pub中。
（值得注意的是，如果你直接打开该文件并且复制到github后，很可能会报错说无效密钥，最好在git中使用

phcopy <~/.ssh/id_rsa.pub

进行复制）
3. 在github右上角都Setting中找到SSH and GPG keys这一项，
把刚刚复制的内容粘贴并选择 Add SSH key即可。
（记得在git bash中使用

ssh -T git@github.com

进行验证）

##node.js下载hexo框架部分

  1. 使用node.js脚手架安装hexo,安装命令为:

    npm install -g hexo-cli

  2. 安装完成之后，cd进入或者在指定目录打开git Bash使用以下指令
    初始化hexo框架

    hexo init

静态部署

hexo g

查看是否部署完成

hexo s

(使用完此命令后会在下面的命令行显式运行的本地地址，一般为http://localhost:4000)

停止运行服务器(以下这条不是指令，是键盘按键)

ctrl + c

绑定本地hexo与github绑定部分

  1. 在安装hexo的根目录（也就是你输入hexo init的目录下）打开_config.yml文件，直接划到该文件最底部，输入以下内容

    deploy:
    type: git
    repository: #你的仓库地址，在github创建仓库后，复制该地址填入此处
    branch: master #有些版本是master,有些是main,主要看你仓库的根分支是什么，当然，如果不行按顺序尝试。

  2. 在hexo的根目录下启动git bash，输入以下指令

    npm install hexo-deployer-git –save

然后依次输入以下指令

hexo clean #清除缓存
hexo g #生成静态网页，generate，比如将source的内容生成到public中
hexo d 生成网站静态页面并且部署到仓库中

  1. 进入你的github仓库，将目的仓库地址复制，删去其中的github.com\即可进入部署的博客中。

hexo的一些基本操作：

  1. next主题安装，进入根目录的themes文件夹中，使用git bash输入

    git clone https://github.com/theme-next/hexo-theme-next themes/next

  2. 生成博文:使用以下指令:

    hexo new [layout] title

  3. source下还可以创建相应的文件，不过需要在.config也需要进行相应的修改。
